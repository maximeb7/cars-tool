## Installation
```bash
- composer install
```
```bash
- npm install
```
- Run the migrations and seeds


## Configuration 

- Don't forget to add a .env file with your database configuration. You may use MySql or Postgre or what you prefer


## Start Project
- Run front-end on your local server typing : 
  ```bash 
  npm run dev
  ```
- Run back-end using : 
```bash
- php artisan serve
```
